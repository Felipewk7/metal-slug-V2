# 📑 GDO: Project Neon Justice (Metal Slug Style)

Welcome to **Project Neon Justice**, a high-octane 2D side-scrolling run 'n gun game built with Vanilla JavaScript and Canvas.

## 🚀 How to Run Locally

1.  Open the project folder in your preferred code editor.
2.  If you have Node.js installed, run:
    ```bash
    npm install
    npm run dev
    ```
3.  Alternatively, you can use an extension like **Live Server** in VS Code to open `index.html`.

## 🎮 Controls
- **[A] / [D] / [Arrows]**: Move Left / Right
- **[Shift]**: Jump
- **[Space]**: Shoot (Hold for Machine Gun)

## 🛠️ Deployment to GitHub Pages

1.  **Repository Setup**:
    - In GitHub Desktop, you should see these files in your `metal slug V2` repository.
    - Commit and Push the changes.
2.  **GitHub Settings**:
    - Go to your repository on GitHub.
    - Click **Settings** > **Pages**.
    - Under **Build and deployment**, select `main` (branch) and `/ (root)` (folder).
    - Save and wait a few minutes.
3.  Your game will be live at `https://[your-username].github.io/metal-slug-v2/`.
